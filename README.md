# Eventmate
 Last-Minute Event Buddy  A web platform that helps users find companions for events in real-time.   Users can create events, discover nearby activities, and connect with verified people through chat.   "Never go alone again."
# 🎟️ Last-Minute Event Buddy

## 📌 Problem
People hesitate to attend events alone, which leads to missed opportunities and social isolation.

## 💡 Solution
This platform allows users to find event companions instantly by connecting with verified users attending similar events.

---

## 🚀 Features
- 🎟️ Event creation and listing
- 🔥 Swipe-based event discovery
- 💬 Real-time chat (AJAX)
- 🔐 User authentication system
- 📍 Location view using maps

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite
